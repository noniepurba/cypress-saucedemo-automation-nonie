# SauceDemo Cypress Automation

This project contains automated test scripts for the SauceDemo website using Cypress.

## Website
https://www.saucedemo.com

## Prerequisites
- Node.js
- npm
- Google Chrome
- Cypress

## Installation

Install project dependencies:

npm install

## Running Tests

Open Cypress Test Runner:

npx cypress open

Run tests in headless mode:

npx cypress run

## Test Scenarios
- Login with valid credentials
- Login failure scenario
- Add product to cart
- Checkout flow
- Logout functionality
